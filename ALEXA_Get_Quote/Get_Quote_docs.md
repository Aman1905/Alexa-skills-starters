# ✨ Get Quote - Alexa Skill ✨ 
This skill speaks out a random inspirational quote from a list of quotes, each time user speaks/types "hello".

## **Skill Invocation -**
Speak/Type :
- "Get Quote"
- "Alexa, open Get Quote"
- "Alexa, play Get Quote"

## **Getting a new quote -**
Speak/Type :
- "Hello"
